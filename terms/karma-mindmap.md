═══════════════════════════════════════════════════════════════════════════════════
ОСНОВА: ЧТО ТАКОЕ КАРМА
═══════════════════════════════════════════════════════════════════════════════════

КАРМА = всё, что вы делаете, говорите, думаете
    |
    ├── Записывается в подсознании как СЕМЕНА
    ├── Семена РАСТУТ и СОЗРЕВАЮТ
    ├── Прорываются наружу → определяют ВОСПРИЯТИЕ
    └── НИЧЕГО НЕ ПРОПАДАЕТ бесследно

Сознание = жёсткий диск с безграничной памятью


═══════════════════════════════════════════════════════════════════════════════════
ОСНОВА: ДВА УРОВНЯ РЕАЛЬНОСТИ
═══════════════════════════════════════════════════════════════════════════════════

КАЖЕТСЯ ИСТИННЫМ (поверхность)           ЯВЛЯЕТСЯ ИСТИННЫМ (глубина)
         |                                        |
    Ключ → Машина                           Аккумулятор → Машина
    Звонок → Продажа                        Карма → Продажа
    Действие → Результат                    Семена → Результат
         |                                        |
    Видимый уровень                         Кармический уровень
    Иногда работает                         ВСЕГДА работает


═══════════════════════════════════════════════════════════════════════════════════
ОСНОВА: ПРИНЦИП ЭХА
═══════════════════════════════════════════════════════════════════════════════════

Другие люди = стены пещеры для эха
    |
    ├── Без других → нет эха
    ├── Без эха → нет семян
    └── Без семян → нет успеха

Дать другим → Получить себе → Вернётся эхом

8 ПРАВИЛ КАРМИЧЕСКОГО МЕНЕДЖМЕНТА

├─────────────────────────────────────────────────────────────────────────────┤

│ • Если иногда работает, иногда нет → НЕ РАБОТАЕТ                            │
│ • Шанс на успех = 100% (при правильном подходе)                             │

└─────────────────────────────────────────────────────────────────────────────┘
                                     ↓
┌─────────────────────────────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────────────────────────────┤
│ • Видимая причина ≠ истинная причина                                        │
│ • Карма = причина причин (под капотом)                                      │
│ • Что посеешь → то пожнёшь                                                  │
│ • Ищите глубинный уровень                                                   │
└─────────────────────────────────────────────────────────────────────────────┘
                                     ↓
┌─────────────────────────────────────────────────────────────────────────────┐

│ 4 ГРУППЫ:                                                                   │
│  1. Коллеги/подчинённые → сделать героями                                   │
│  2. Клиенты → быть одержимым их успехом                                     │
│  3. Поставщики → проявлять личный интерес                                   │
│  4. Весь мир → создавать ценность для всех                                  │
│                                                                             │
│ Сделайте ИХ успешными → станете успешны ВЫ                                  │
└─────────────────────────────────────────────────────────────────────────────┘
                                     ↓
┌─────────────────────────────────────────────────────────────────────────────┐
│ ПРАВИЛО #4: НАЧНИТЕ С САМОГО СЕБЯ                                           │
├─────────────────────────────────────────────────────────────────────────────┤
│ • "ТЫ" = "Я" (граница иллюзорна)                                            │
│ • Истинное Я = огромно (включает всех)                                      │
│ • Конкуренты = тоже партнёры                                                │
│ • Материнская ответственность за всех                                       │
│ • Чистое сознание = главный инструмент                                      │
└─────────────────────────────────────────────────────────────────────────────┘
                                     ↓
┌─────────────────────────────────────────────────────────────────────────────┐
│ ПРАВИЛО #5: ПРЕКРАТИТЕ ПРИНИМАТЬ РЕШЕНИЯ                                    │
├─────────────────────────────────────────────────────────────────────────────┤
│ • Решения → из неуверенности                                                │
│ • Решения → порождают новые решения                                         │
│ • Успешные люди → действуют по ИНСТИНКТУ                                    │
│ • Инстинкт появляется → при заряженных действиях                            │
│ • Не решать → а ЧУВСТВОВАТЬ правильное                                      │
└─────────────────────────────────────────────────────────────────────────────┘
                                     ↓
┌─────────────────────────────────────────────────────────────────────────────┐
│ ПРАВИЛО #6: ЗАГРУЗИТЕ ВАШ СТЕПЛЕР                                           │
├─────────────────────────────────────────────────────────────────────────────┤
│ Степлер работает если:                                                      │
│  • ЗАРЯЖЕН → всегда работает                                                │
│  • НЕ заряжен → никогда не работает                                         │
│  • Цвет НЕ ВАЖЕН                                                            │
│                                                                             │
│ Так же с действиями:                                                        │
│  • Заряжены кармой → всегда сработают                                       │
│  • Не заряжены → провалятся в любом случае                                  │
│                                                                             │
│ КАК ЗАРЯДИТЬ: Каждый день делать 4 действия для партнёров                   │
└─────────────────────────────────────────────────────────────────────────────┘
                                     ↓
┌─────────────────────────────────────────────────────────────────────────────┐
│ ПРАВИЛО #7: СКАЧИТЕ НА ПРОБЛЕМАХ К ВЕРШИНАМ                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│ Проблемы = ДРУЗЬЯ (показывают где карма)                                    │
│                                                                             │
│ РАБОТА С ПРОБЛЕМАМИ:                                                        │
│  1. Определить тип проблемы                                                 │
│  2. Найти кармическую причину                                               │
│  3. Закрыть кармический кран                                                │
│  4. Создать резонанс (делать противоположное)                               │
│                                                                             │
│ НЕ РЕАГИРОВАТЬ естественно (насилие → насилие)                              │
│ РАДОВАТЬСЯ проблемам и фокусироваться на них                                │
└─────────────────────────────────────────────────────────────────────────────┘
                                     ↓
┌─────────────────────────────────────────────────────────────────────────────┐
│ ПРАВИЛО #8: ИНВЕСТИРУЙТЕ КАРМУ ВНОВЬ                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│ • Использованная карма → умирает                                            │
│ • Откладывать 10% урожая → для нового посева                                │
│                                                                             │
│ ПОСЛЕ УСПЕХА:                                                               │
│  • Команда → ужин в ресторане (10% бонуса)                                  │
│  • Поставщики → подарки + новые заказы                                      │
│  • Клиенты → дополнительная ценность                                        │
│  • Мир → пожертвования                                                      │
│                                                                             │
│ РЕЗУЛЬТАТ: Восходящая спираль вместо упадка                                 │
└─────────────────────────────────────────────────────────────────────────────┘

│     ОБРАЗ              │              ЗНАЧЕНИЕ                     │

│ Мать четырёх сыновей    │ Ответственность за партнёров              │

Я отдельно от других            ←→      Я = все вокруг
Конкуренты = враги              ←→      Конкуренты = партнёры
Проблемы = угроза               ←→      Проблемы = учителя
Успех = удача                   ←→      Успех = карма
Фокус на своём успехе           ←→      Фокус на успехе других
Видимый уровень                 ←→      Истинный уровень
Кажется истинным                ←→      Является истинным
