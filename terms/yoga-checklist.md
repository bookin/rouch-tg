# Чек-лист практики Тибетской Сердечной Йоги

## Подготовка к практике

### За 3 часа до занятия
- [ ] Не есть тяжелую пищу
- [ ] Можно пить воду и сок

### Перед началом (за 10-15 минут)
- [ ] Опорожнить кишечник и мочевой пузырь
- [ ] Надеть свободную одежду из натуральных тканей
- [ ] Приготовить коврик + покрывало
- [ ] Проветрить помещение (не холодно!)
- [ ] Убрать лишние предметы из комнаты
- [ ] Выключить телефон/уведомления
- [ ] Приготовить таймер (или часы)
- [ ] Иметь в виду конкретного человека для практики

### Выбор человека для практики
- [ ] Кто-то, кому сейчас трудно (друг, родственник)
- [ ] Четко представляю его лицо и ситуацию
- [ ] Знаю, какая боль его мучает
- [ ] Готов принять его страдание

---

## Во время практики (30 минут)

### УПРАЖНЕНИЕ 1: Алмаз в розе (2 мин)
- [ ] Сесть удобно, ноги скрещены
- [ ] Ладони у сердца, большие пальцы между ладонями
- [ ] Прикрыть глаза
- [ ] Сфокусироваться между бровями → выше → вглубь
- [ ] Перенести внимание в сердце
- [ ] Визуализировать красную розу с алмазом
- [ ] Петь: "Ом мани падме хум" (или альтернативу)
- [ ] Удерживать образ до конца упражнения

✓ **Проверка**: Чувствую ли я расслабление в груди?

---

### УПРАЖНЕНИЕ 2: Совершенная десятка (2 мин)
- [ ] Остаться сидеть удобно
- [ ] Руки на коленях, большой + указательный пальцы соединены
- [ ] Спина прямая, плечи расслаблены
- [ ] Взгляд вниз → медленно прикрыть глаза
- [ ] Долгий медленный выдох через нос
- [ ] Затем тихий вдох (= 1 цикл)
- [ ] Считать циклы, фокус на носу
- [ ] Дойти до 10 без отвлечений
- [ ] Начать сначала, если сбился

✓ **Проверка**: 
- Дыхание начинается с выдоха?
- Ум спокоен?

---

### УПРАЖНЕНИЕ 3: Разгоняем тьму (3 мин)
**ПРИНЯТИЕ**

- [ ] Продолжать сидеть
- [ ] Фокус: межбровье → выше → вглубь → сердце
- [ ] Алмаз и роза всё еще там
- [ ] 5 циклов подготовительного дыхания

**Визуализация принятия**:
- [ ] Представить дорогого страдающего человека
- [ ] Увидеть помещение, где он находится
- [ ] Мысленно сесть перед ним (невидимый)
- [ ] Вся его боль = черное облачко в сердце
- [ ] Конкретно почувствовать его страдание
- [ ] Твердо решить забрать боль на себя
- [ ] Несколько вдохов: облако поднимается
- [ ] Черная струйка выходит из его ноздрей
- [ ] Собирается в облако перед вашим носом
- [ ] Пауза: приготовиться к финальному вдоху
- [ ] Взглянуть на свой алмаз (он разрушит всё)
- [ ] ВДОХ: черная струя входит в ваш нос
- [ ] Облако касается алмаза
- [ ] **ВЗРЫВ ЗОЛОТОГО СВЕТА** в сердце
- [ ] Тьма полностью уничтожена
- [ ] Остался только белый дым
- [ ] Увидеть человека счастливым и свободным
- [ ] Радоваться его освобождению

✓ **Проверка**: 
- Не осталось ли тьмы во мне?
- Вижу ли я полное исчезновение страдания?

---

### УПРАЖНЕНИЕ 4: Даруем вместе с солнцем (5 мин)
**"Поклон солнцу" + ДАРОВАНИЕ**

**Подготовка**:
- [ ] Встать перед ковриком, ноги вместе
- [ ] Вдох, сказать: "Я дарую тебе умение отдавать"
- [ ] Ладони у груди, закрыть глаза
- [ ] 5 циклов дыхания с ароматом розы
- [ ] Представить световой мост к человеку

**Алмаз неба** (5 циклов):
- [ ] ВЫДОХ: Ладони у сердца
- [ ] ВДОХ: Руки вверх, прогиб в области СЕРДЦА (не шея!)
- [ ] Взгляд прямо
- [ ] Удерживать 5 циклов

**Поклон солнцу** (выполнить с обеих ног):
- [ ] ВЫДОХ: Наклон вперед, грудь к ногам
- [ ] ВДОХ: Левая нога назад (выпад)
- [ ] ВЫДОХ: Обе ноги назад, таз вверх
- [ ] ВДОХ: Упор лежа (прямая линия)
- [ ] ВЫДОХ: Опуститься на пол
- [ ] ВДОХ: Прогиб назад (от СЕРДЦА!)
- [ ] ВЫДОХ: Таз вверх (5 циклов)
- [ ] ВЫДОХ: Выпад левой ногой
- [ ] ВДОХ: Стоя в наклоне
- [ ] ВЫДОХ: Обхватить лодыжки, грудь к ногам
- [ ] ВДОХ: Ладони вместе, вверх, прогиб
- [ ] ВЫДОХ: Ладони к сердцу

**С каждым ВЫДОХОМ**:
- [ ] Аромат розы выходит из ноздрей
- [ ] Входит в нос дорогого человека
- [ ] Его алмаз начинает сверкать
- [ ] Световой мост между сердцами
- [ ] Он получает дар ЩЕДРОСТИ

✓ **Проверка**:
- Прогиб в сердце, не в шее?
- Дарю ли с каждым выдохом?
- Лицо расслаблено?

---

### УПРАЖНЕНИЕ 5: Доброта с запада (4 мин)

**Подготовка**:
- [ ] Сесть, ноги вперед, ладони в пол
- [ ] Вдох, сказать: "Я посылаю тебе доброту"
- [ ] 5 циклов дыхания + свет

**Западная растяжка**:
- [ ] ВЫДОХ: Наклон, взяться за лодыжки/пальцы ног
- [ ] ВДОХ: Прогиб назад, открыть сердце
- [ ] ВЫДОХ: Грудь на ноги
- [ ] ВДОХ: Держать, смотреть на пальцы ног
- [ ] 5 циклов в позе

**Алмазное колесо** - Скрутки:
- [ ] Скрестить ноги, пальцы в замок за спиной
- [ ] ВЫДОХ: Поворот влево (5 циклов)
- [ ] ВЫДОХ: Вернуться в центр
- [ ] ВЫДОХ: Поворот вправо (5 циклов)
- [ ] Скручивание в области СЕРДЦА!

**Наклоны в стороны**:
- [ ] ВДОХ: Руки над головой в замке
- [ ] ВЫДОХ: Наклон влево (5 циклов)
- [ ] ВДОХ: Центр
- [ ] ВЫДОХ: Наклон вправо (5 циклов)

**Наклон вперед**:
- [ ] ВЫДОХ: Руки за спиной, наклон вперед (5 циклов)
- [ ] Лоб параллельно полу, спина прямая

**Прогиб назад**:
- [ ] ВЫДОХ: Руки за спиной, прогиб (5 циклов)
- [ ] Взгляд в небо

**С каждым ВЫДОХОМ**: Дарю ДОБРОТУ

✓ **Проверка**: Чувствую ли растяжение в пояснице и сердце?

---

### УПРАЖНЕНИЕ 6: Царь Терпения (4 мин)

**Подготовка**:
- [ ] Сесть прямо, ноги вперед
- [ ] Вдох, сказать: "Я посылаю тебе терпение"
- [ ] 5 циклов дыхания + свет

**Царь рыб** (с обеих сторон):
- [ ] ВЫДОХ: Левая стопа у правого колена, правая нога согнута
- [ ] ВДОХ: Правый локоть к левому колену
- [ ] ВЫДОХ: Поворот влево, взгляд через плечо
- [ ] 5 циклов
- [ ] Повторить вправо

**Крюк**:
- [ ] ВДОХ: Пальцы сцеплены перед сердцем
- [ ] ВЫДОХ: Резко вперед, вытянуться
- [ ] ВЫДОХ: Поворот влево (3 цикла)
- [ ] ВЫДОХ: Согнуть левую руку, еще дальше (3 цикла)
- [ ] Повторить вправо

**С каждым ВЫДОХОМ**: Дарю ТЕРПЕНИЕ

✓ **Проверка**: Скручивание от сердца?

---

### УПРАЖНЕНИЕ 7: Лук и Стрела радости (3 мин)

**Подготовка**:
- [ ] Сесть прямо
- [ ] Сказать: "Я посылаю тебе радость"
- [ ] 5 циклов дыхания + свет

**Большой лук** (2 раза):
- [ ] ВЫДОХ: Лечь на живот
- [ ] ВЫДОХ: Согнуть ноги, схватить лодыжки
- [ ] ВДОХ: Потянуться, оторвать грудь и бедра
- [ ] 5 циклов
- [ ] ВЫДОХ: Опуститься, отдых

**Стрела**:
- [ ] ВДОХ: На колени, пальцы ног в пол, сесть на пятки
- [ ] ВЫДОХ: Ладони на пол, прогиб, голова назад
- [ ] ОТКРЫТЬ РОТ, растянуть лицо (поза Льва)
- [ ] 5 циклов дыхания через нос
- [ ] ВЫДОХ: Голову вниз, спина колесом
- [ ] 5 циклов с резким "ХА" на выдохе

**С каждым ВЫДОХОМ**: Дарю РАДОСТНЫЕ УСИЛИЯ

✓ **Проверка**: 
- Растяжение лица?
- Резкий выдох "ХА"?

---

### УПРАЖНЕНИЕ 8: Безмолвие с небес (3 мин)

**Подготовка**:
- [ ] Сесть
- [ ] Вдох, сказать: "Я посылаю тебе спокойствие"
- [ ] 5 циклов дыхания + свет

**Поза для всего тела**:
- [ ] ВЫДОХ: Лечь на спину
- [ ] ВДОХ: Ноги за голову на пол
- [ ] Сомкнуть пальцы, свести лопатки
- [ ] Согнуть руки, ладони в спину
- [ ] ВДОХ: Ноги вверх перпендикулярно
- [ ] Вес на плечах, НЕ на шее!
- [ ] Взгляд за кончик носа
- [ ] 15 циклов (позже 25)
- [ ] ВЫДОХ: Ноги за голову (5 циклов)
- [ ] ВЫДОХ: Медленно опуститься
- [ ] 5 вздохов лежа

**Тряпичная кукла**:
- [ ] Встать, ноги вместе
- [ ] Пальцы в замок за спиной
- [ ] ВДОХ: Прогиб от СЕРДЦА
- [ ] Голова свободно назад
- [ ] Потянуть руки к полу

**С каждым ВЫДОХОМ**: Дарю СПОКОЙСТВИЕ

✓ **Проверка**: 
- Вес на плечах?
- Несколько вздохов перед подъемом?

---

### УПРАЖНЕНИЕ 9: Смерть неведения (4 мин)

**Поза мертвого человека** (2 мин):
- [ ] Лечь на спину
- [ ] Руки и ноги расслаблены
- [ ] Ладони вверх
- [ ] Вытянуть шею
- [ ] Закрыть глаза
- [ ] Можно укрыться одеялом
- [ ] Медленные вздохи
- [ ] Вес тела уходит в пол
- [ ] Напряжение утекает через пол

**Дарование мудрости** (2 мин):
- [ ] Вдох, сказать: "Я посылаю тебе мудрость"
- [ ] Оставаться неподвижным
- [ ] 2 минуты посылать мудрость

**Растяжка-встряхивание**:
- [ ] Медленно шевелить руками и ногами
- [ ] Встряхивать конечности
- [ ] Поворачивать голову
- [ ] Растягивать пальцы рук
- [ ] Массировать ладони
- [ ] Растирать руки

**Завершение**:
- [ ] Медленно повернуться набок
- [ ] Помогая руками, сесть
- [ ] Спокойно посидеть
- [ ] Вспомнить шесть даров
- [ ] Порадоваться своим усилиям
- [ ] Увидеть световой мост
- [ ] На миг: бесконечные мосты ко всем существам

✓ **Проверка**: Чувствую ли расслабление во всем теле?

---

## После практики

### Сразу после (30 минут)
- [ ] Не есть
- [ ] Можно пить воду/сок
- [ ] Спокойно посидеть
- [ ] Не бросаться в активность

### В течение дня
- [ ] Держать спину прямой
- [ ] Подбирать живот
- [ ] Улыбаться
- [ ] Долгие медленные выдохи
- [ ] Практика "Дарование-принятие" с незнакомцами
- [ ] В пробке, в метро, в офисе
- [ ] Невидимо для других

---

## Мысленный вопросник (во время практики)

### Постоянно проверять:
1. [ ] **Дыхание**: Начинается с выдоха?
2. [ ] **Выдох**: Долгий и медленный?
3. [ ] **Лицо**: Веселое и расслабленное? Улыбаюсь?
4. [ ] **Взгляд**: Фокусируюсь на точке?
5. [ ] **Растяжка**: Растягиваю внутренние каналы?
6. [ ] **ГЛАВНОЕ**: Дарую ли с каждым выдохом?

- [ ] Полные 30 минут каждый раз
- [ ] Один выходной в одни день недели

### Наблюдения

## Признаки правильной практики

### Физические
- [ ] Легкая боль в мышцах первые недели (нормально!)
- [ ] Постепенное увеличение гибкости
- [ ] Дыхание стало глубже и ровнее
- [ ] Улучшение пищеварения
- [ ] Регулярное опорожнение кишечника
- [ ] Улучшение сна
- [ ] Больше энергии в течение дня

### Ментальные
- [ ] Спокойствие ума после практики
- [ ] Чувство легкости
- [ ] Радость от практики
- [ ] Желание помогать другим
- [ ] Меньше раздражения в течение дня
- [ ] Лучшая концентрация

### Духовные
- [ ] Чувство связи с человеком в практике
- [ ] Искренняя радость его счастью
- [ ] Спонтанное сострадание к незнакомцам
- [ ] Понимание кармы в жизни
- [ ] Замечание созревания семян

## Предостережения (проверить!)

### НЕ делать:
- [ ] ❌ Не форсировать растяжку
- [ ] ❌ Не задерживать дыхание
- [ ] ❌ Не сравнивать себя с другими
- [ ] ❌ Не заниматься на полный желудок
- [ ] ❌ Не пить кофе/чай перед практикой
- [ ] ❌ Не прогибать шею (только область сердца!)
- [ ] ❌ Не прогибать поясницу (только область сердца!)

### При проблемах:
- [ ] Серьезная боль → остановиться
- [ ] Головокружение → медленнее переходы
- [ ] Одышка → ослабить позу
- [ ] Сонливость → приоткрыть глаза
- [ ] Блуждание ума → вернуться к вопроснику
